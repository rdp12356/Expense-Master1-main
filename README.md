# Expense Master 💰

Expense Master is the ultimate modern, open-source, cross-platform expense management application. It is built to seamlessly sync your financial data across Web, iOS, Android, and Desktop from a single bleeding-edge codebase.

This project is built using the "Perfect Stack":
- **Framework**: Expo (React Native) for universal apps.
- **Backend & DB**: Supabase (PostgreSQL + Realtime Sync).
- **Styling**: NativeWind (Tailwind CSS for React Native).
- **Animations**: Reanimated & Moti for 120fps, highly responsive micro-interactions, text overlays, floating elements, and hover effects on web.
- **Internationalization**: Full 195+ World Languages and Currencies natively integrated.

## Features
- 💸 **Universal Transactions**: Manage income & expenses natively on your phone or on the web.
- 🎨 **Premium Aesthetic**: Fluid gradients, custom Google fonts, hover effects, and dark mode out-of-the-box.
- 📊 **Animated Analytics**: Watch your balance adapt instantly.
- 🌍 **Multi-Language**: Built for everyone.
- 🔒 **Privacy First**: Fully open source. Data stored securely in Supabase with RLS.

## How to Run locally
This project uses the Expo universal toolchain:

```bash
cd frontend_expo
npm install
# To run on Web
npx expo start -w
# To run on iOS/Android Simulator
npx expo start
```
